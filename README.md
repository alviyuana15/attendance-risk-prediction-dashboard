# attendance-risk-prediction-dashboard
Interactive dashboard for analyzing employee attendance patterns and predicting lateness risk using data analytics and forecasting techniques.

📌 Background

This project was developed during my professional experience at PT PAL to support HR decision-making through data-driven insights on employee attendance and lateness behavior.

🚀 Key Features
📊 Attendance Analytics
Attendance rate monitoring
Weekly and monthly trends visualization
KPI-based performance tracking
📉 Attendance Forecasting
Future attendance prediction
Risk-based threshold analysis
Early warning system for low attendance periods
🔥 Lateness Risk Analysis
Employee-level risk scoring
Ranking of high-risk employees
Division-based lateness trends
🧠 Predictive Modeling
Forecast lateness probability per employee
Time series analysis for trend detection
Risk classification (Low, Medium, High)
🗺️ Visual Insights
Heatmap of attendance patterns
Trend line analysis across divisions
Distribution analysis of attendance rates
🛠️ Tech Stack
Python
Streamlit
Pandas
NumPy
Plotly / Matplotlib
